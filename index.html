Hoelang duurt het maken van een website?

Het maken van een website kan al binnen 2 weken klaar zijn, afhankelijk van de grootte van de website.

Kan ik zelf de website beheren nadat deze is gebouwd?

Het aanpassen van de website gaat minder makkelijk dan bijvoorbeeld een WordPress of Wix website. Wanneer je een aanpassing wilt, kan ik dit voor een betaalbare prijs voor je doen. Ik kan ook uitleg geven zodat je het zelf kunt aanpassen.

Is de website te gebruiken op mobiel?

Ja, de website werkt op mobiele telefoons, tablets en computers, en op alle formaten daartussen.

Is de website te gebruiken op alle browsers?

Ja, de website werkt op verschillende browsers zoals Google Chrome, Safari, Firefox en Edge.
